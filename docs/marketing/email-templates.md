## 4. Security Best Practices

**Subject:** Keep your VELX safe — VeloxDAG security essentials

**From:** VeloxDAG <hello@veloxdag.com>

---

Hi {{first_name}},

Fair launch puts **you** in custody of your earnings. Please follow these **security best practices** for VeloxDAG mining and wallets.

**Wallet**
- Back up `wallet_*.json` offline; use encryption at rest  
- Never share private keys, seeds, or wallet files — **we will never ask**  
- Verify `velx1` addresses character-by-character before sending  

**Node & RPC**
- Prefer mining on **localhost**; avoid exposing RPC (port 8545) to the public internet  
- If remote RPC is required, firewall by IP and use strong authentication  

**Operational**
- Download binaries only from official links on veloxdag.com / verified GitHub releases  
- Ignore DMs offering “support,” airdrops, or doubling schemes — **scams spike at launch**  

Report suspicious impersonation to hello@veloxdag.com.

Stay safe,
